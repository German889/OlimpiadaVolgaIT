Класс Main запускает метод fileToDBmove() класса Manager.
Класс Manager содержит переменные-строки для хранения данных для:
    открытия исходного файла
    подключения базы данных
Значения этих переменных вводятся через консоль С помощью класса ConsoleInput.

Manager начинает чтение файла с помощью класса DocumentReader. DocumentReader очищает текст файла от ненужных
элементов с помощью класса HTMLDocumentReader. Если потребуется принять файл не html (как в задании), то можно создать
класс для его обработки, затем подключив его к DocumentReader, через метод perform().

HTMLDocumentReader реализует очистку строки от тегов и пунктуации. С некоторыми тегами работает неправильно, считая их
словами, но при необходимости можно заменить другим классом, подключив его через метод perform().

StatisticMaker считывает из подготовленного (очищенного от тегов и пунктуации) txt файла слова, подсчитывает их
количество и частоту повторений. Переписывает в итоговый файл (resultSorted.txt) построчно все слова с указанием
количества их повторений и исходном файле. Все повторения слов удаляются до записи в этот файл.

Класс DBwriter подключается к базе данных MySQL с логином и паролем которые пользователь вводил в консоль.
Далее весь файл resultSorted.txt он построчно в том же порядке переписывает в базу данных.

Класс OutToConsole выводит содержимое файла resultSorted.txt в косоль.

Класс FileRemaker создаёт и удаляет промежуточные файлы: out.txt result.txt resultSorted.txt;
